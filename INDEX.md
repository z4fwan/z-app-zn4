## ğŸ› ï¸ Helper Scripts

### fix-and-start.bat
**Purpose**: Clean install and start  
**Use When**: First time setup or fixing issues  
**What It Does**:
- Removes all node_modules
- Cleans npm cache
- Installs all dependencies
- Starts both servers

```bash
fix-and-start.bat
```

---

### install-all.bat
**Purpose**: Install dependencies  
**Use When**: First time or after pulling updates  
**What It Does**:
- Installs root dependencies
- Installs backend dependencies
- Installs frontend dependencies

```bash
install-all.bat
```

---

### start-dev.bat
**Purpose**: Start development servers  
**Use When**: Daily development  
**What It Does**:
- Starts backend server (port 5001)
- Starts frontend server (port 5173)
- Runs both concurrently

```bash
start-dev.bat
```

---

### kill-port-5001.bat
**Purpose**: Free up port 5001  
**Use When**: Port is already in use  
**What It Does**:
- Finds process using port 5001
- Kills the process
- Frees the port

```bash
kill-port-5001.bat
```

---

### push-to-github.bat
**Purpose**: Push code to GitHub  
**Use When**: Committing changes  
**What It Does**:
- Adds all changes
- Commits with message
- Pushes to GitHub

```bash
push-to-github.bat
```
